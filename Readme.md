# lc

## Description

输出给定目录下文件夹或文件的数量

## install

```bash
    npm install lcx -g
```

## usage

```bash
  Usage: lc [options] [dir]

  list directory contents count


  Options:

    -V, --version    output the version number
    -d, --directory  output directory folder count
    -f, --file       output directory file count
    -h, --help       output usage information
```
